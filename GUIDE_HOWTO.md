# 攻略記事の追加方法

## 概要

攻略記事は `client/src/lib/data.ts` の `SAMPLE_GUIDES` 配列に追加することで管理します。
Markdownで本文を書けるので、見出し・箇条書き・太字などが使えます。

---

## 新しい記事を追加する手順

### 1. `client/src/lib/data.ts` を開く

### 2. `SAMPLE_GUIDES` 配列に新しいオブジェクトを追加する

```typescript
{
  id: 'guide-XXX',           // ユニークなID（連番推奨）
  title: '記事タイトル',
  category: 'キャラ評価',    // カテゴリ（下記から選択）
  tags: ['タグ1', 'タグ2'],  // 検索・フィルター用タグ
  summary: '記事の概要（一覧ページに表示される短い説明）',
  content: `
## 見出し2

本文をMarkdownで書きます。

### 見出し3

- 箇条書き1
- 箇条書き2

**太字テキスト**

| 列1 | 列2 |
|-----|-----|
| データ1 | データ2 |
  `,
  author: 'まつモン',
  publishedAt: '2026-03-10',  // 公開日
  updatedAt: '2026-03-10',    // 更新日
  isOfficial: true,           // 管理者記事はtrue
}
```

---

## カテゴリ一覧

| カテゴリ | 説明 |
|---------|------|
| `キャラ評価` | 副将の性能評価 |
| `ダメージ計算` | ダメージ計算式・解説 |
| `初心者ガイド` | 始め方・基本知識 |
| `イベント攻略` | イベント攻略情報 |
| `育成・強化` | 育成・強化方法 |
| `その他` | その他の情報 |

---

## キャラ評価ランクを追加する

`client/src/pages/Characters.tsx` の `CHARACTERS` 配列に追加します：

```typescript
{
  id: 'c-XXX',
  name: 'キャラクター名',
  rank: 'S',           // S+, S, A+, A, B, C から選択
  type: '物理',        // 物理 or 魔法
  tags: ['SSR', 'アタッカー'],
  guideId: 'guide-XXX', // 関連する攻略記事のID（なければundefined）
}
```

---

## Markdownで使える記法

```markdown
## 見出し2（金色で表示）
### 見出し3

**太字**（金色で表示）
*斜体*

- 箇条書き
  - ネスト

1. 番号付きリスト

`コード`（朱色で表示）

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
```
